
# StocksProject

## To run
- Clone Repository
 ```
 git clone https://github.com/svengia25/StocksProject.git
 ```
 ### React App
- cd StocksProject
- run npm i
- npm start

 ### Node Server
- cd StocksProject
- cd server
- add .env File with API_KEY for https://www.alphavantage.co/
- run npm i
- npm start

Becareful of API limits
